
# CustNote :fire:

Blazing fast and highly customizable note-taking web application that will fit your needs!

## Run Locally

Clone the project

```bash
  git clone https://github.com/AdrakPro/cust-note/
```

Go to the project directory

```bash
  cd cust-note
```

Install dependencies

```bash
  yarn install
```

Start developing

```bash
  yarn dev
```
